## About the Project
Bookstore is a full-fledged platform for managing a bookshop. The project includes:
- Backend on Kotlin.
- Frontend (website).
- Android mobile application.
- Containerization using Docker.
- Orchestration with Kubernetes.
- Monitoring via Grafana.
